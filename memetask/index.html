Текстовый файл состоит не более чем из 106 символов латинского алфавита (A, B, C, ...). Определите максимальное количество идущих подряд символов, среди которых символ Z или V встречается не более 228 раз.

Файл 24_1.txt

#решение методом двух указателей

s = open('24_1.txt').readline()

k, l, mx = 0, 0, float('-inf')

for r in range(len(s)):

  k += s[r] == 'Z' or s[r] == 'V'

  while k > 228:

    k -= s[l] == 'Z' or s[l] == 'V'

    l += 1

  mx = max(mx, r-l+1)

print(mx)
