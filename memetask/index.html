s = open('24_1.txt').readline()

k, l, mx = 0, 0, float('-inf')

for r in range(len(s)):

  k += s[r] == 'Z' or s[r] == 'V'

  while k > 228:

    k -= s[l] == 'Z' or s[l] == 'V'

    l += 1

  mx = max(mx, r-l+1)

Исполнитель Абубачир преобразует число на экране. У него есть 4 команды, которым присвоены номера:

1. Прибавить 69

2. Умножить на 228

3. Вычесть 666 (отрицательные числа допускаются)

4. Поделить на 13 (если число кратно 13)

Сколько различных чисел может получиться из числа 404 с помощью программы, которая содержит ровно 11 команд?

a = set()

def f(x, k):

  if k == 11:

    a.add(x)

    return

  f(x+69, k+1)

  f(x*228, k+1)

  f(x-666, k+1)

  if x%13==0:

    f(x//13, k+1)

f(404, 0)

print(len(a))
